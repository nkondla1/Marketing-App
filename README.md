# Marketing E-commerce App (Next.js)

A sample marketing-focused e-commerce storefront using Next.js (App Router) with:

- product catalog for marketing assets and services
- shopping cart with quantity management
- checkout flow placeholder
- responsive UI with Tailwind CSS

## Usage

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open `http://localhost:3000`

4. Explore the product catalog and add items to the cart.

## Project Files

- `app/page.tsx`: main UI and cart logic
- `app/globals.css`: base styling

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Notes

- This is a starter implementation. Add integrations for payments, authentication, and backend APIs as needed.
